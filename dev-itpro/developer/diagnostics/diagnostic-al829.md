# Compiler Error AL0829

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

Symbol of kind {0} '{1}' cannot be moved to its own app.


## Description
MovedTo/MovedFrom property cannot have Guid that matches its own AppId.
