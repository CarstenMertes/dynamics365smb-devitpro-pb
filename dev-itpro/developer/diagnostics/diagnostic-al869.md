# Compiler Warning AL0869

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

'{0}' '{1}' was expected to be found in app '{2}' but was instead found in app '{3}', which is propagating its dependencies to '{2}'.


## Description
The symbol's reference specification points it to be in an app, but it wasn't found there. Instead, it was found using the reference propagated.
