# Compiler Error AL0825

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

Field '{0}' cannot be moved to {1} '{2}' because '{2}' is marked as Moved.


## Description
A field cannot be moved to a symbol that is already marked as Moved.
