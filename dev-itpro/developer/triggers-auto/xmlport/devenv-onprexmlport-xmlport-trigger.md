# OnPreXmlPort (Xml Port) Trigger
> **Version**: _Available or changed with runtime version 1.0._

Runs after the table views and filters are set and before the XMLport is run.


## Syntax
```AL
trigger OnPreXmlPort()
begin
    ...
end;
```
