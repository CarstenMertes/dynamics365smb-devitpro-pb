# Page.Close() Method
> **Version**: _Available or changed with runtime version 1.0._

Closes the current page.


## Syntax
```AL
 Page.Close()
```
## Parameters
*Page*  
&emsp;Type: [Page](page-data-type.md)  
An instance of the [Page](page-data-type.md) data type.
