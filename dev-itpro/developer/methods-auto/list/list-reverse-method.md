# List.Reverse() Method
> **Version**: _Available or changed with runtime version 1.0._

Reverses the order of the elements in the entire List.


## Syntax
```AL
 List.Reverse()
```

## Parameters
*List*  
&emsp;Type: [List](list-data-type.md)  
An instance of the [List](list-data-type.md) data type.
