# Xmlport.Run() Method
> **Version**: _Available or changed with runtime version 1.0._

Loads and executes the XmlPort that you specify.


## Syntax
```AL
 Xmlport.Run()
```
## Parameters
*Xmlport*  
&emsp;Type: [Xmlport](xmlport-data-type.md)  
An instance of the [Xmlport](xmlport-data-type.md) data type.
