# CustomActionType Property
> **Version**: _Available or changed with runtime version 10.0._

Sets the type of the custom action.

## Applies to
-   Page Custom Action

## Property Value

|Value|Description|
|-----------|---------------------------------------|
|**Flow**|An action that can trigger a Power Automate Flow.|
