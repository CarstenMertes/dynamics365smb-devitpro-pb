# HelpLink Property
> **Version**: _Available or changed with runtime version 1.0._

Specifies the help link to show when the user presses Help in the UI.

## Applies to
-   Page
-   Request Page
-   Query

[//]: # (IMPORTANT: END>DO_NOT_EDIT)

## Syntax

```AL
HelpLink = 'https://www.my-help-link-page.com';
```
