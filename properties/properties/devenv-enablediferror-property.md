# EnabledIfError Property

Sets a value that specifies whether an action is enabled or disabled if it enters an error state.  
  
## Applies to  
  
- Actions  
- Pages  
  
## Property Value

**True**/**false**  
 
## Syntax

```AL
EnabledIfError = true;
```
