# TextType Property

Sets the type of text this element or attribute contains.  
  
## Applies to  

- Text elements in XMLports  
- Text attributes in XMLports
  
## Property Value  

**BigText** if you want to display large text ; otherwise, **Text**. The default is **Text**.  
 
## Syntax

```AL
TextType = BigText;
```
