# ModifyAllowed Property

Sets the value to determine whether users can modify records while using this page.  
  
## Applies to  
  
- Pages  
  
## Property Value

**True** if you want to allow the users to modify records; otherwise, **false**. The default is **true**. 

## Syntax

```AL
ModifyAllowed = false;
```
