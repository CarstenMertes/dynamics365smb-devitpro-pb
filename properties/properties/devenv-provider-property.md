# Provider Property

Sets the provider for a FactBox. This property enables you to create a link from a Repeater or any other type of control to a Factbox. It could also be used to link two FactBoxes.
 
## Applies to  
  
- Page system parts
- Page parts

## Syntax

```AL
Provider = PartName;
```
