# WordMergeDataItem Property

Specifies the name of the root-level **DataItem** that is used for generating separate reports for multiple records.

## Applies to  
  
- Reports  

## WordMergeDataItem Data Type

String

## Syntax

```AL
 WordMergeDataItem = Header;
```
