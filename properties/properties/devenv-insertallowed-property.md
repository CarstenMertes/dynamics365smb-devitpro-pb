# InsertAllowed Property

Sets a value to specify whether users can add records while using a page.  
  
## Applies to  
  
- Pages  
  
## Property Value  

**True** if you want to allow users to add records; otherwise, **false**. The default value is **true**.  

## Syntax

```AL
InsertAllowed = false;
```
