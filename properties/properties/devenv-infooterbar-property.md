# InFooterBar Property

Sets whether an action should be viewed as an exit action in the lower part of the page.  
  
## Applies to  
  
- Actions  
- Pages  
 
## Property value

**True** if the action is viewed as an exit action, otherwise **false**. The default is **false**.

## Syntax

```AL
InFooterBar = false;
```
